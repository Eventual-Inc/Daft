Daft Documentation
==================

Daft is a **fast and scalable Python dataframe** for complex data and machine learning workloads.

Installing Daft
---------------

To install Daft, run this from your terminal:

``pip install getdaft``

Learn about other more advanced installation options in our :doc:`Installation Guide <install>`.
