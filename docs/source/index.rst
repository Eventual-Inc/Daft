Daft API Reference
==================

Daft is the open-sourced distributed Dataframe framework for querying, processing and building Complex Data applications.

This is the API documentation for users of Daft's DataFrame. For higher-level usage examples, please use the `Daft User Docs <https://docs.getdaft.io/daft/>`_.
