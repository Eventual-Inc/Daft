# Developing on Daft

1. [Install Poetry](https://python-poetry.org/docs/#installation)
2. Init your python environment
    - `poetry install`
3. Build Extensions locally
    - `poetry run build_inplace`
4. Run tests
    - `poetry run pytest`
5. Run type checking
    - `poetry run mypy`
6. Run any other script
    - `poetry run CMD`
7. Add package
    - `poetry add PACKAGE`
8. Lock env
    - `poetry lock`
